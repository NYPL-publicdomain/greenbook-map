About This Project Map Another Trip

The Green Book was a travel guide published between 1936 and 1966 that listed hotels, restaurants, bars, gas stations, etc. where black travelers would be welcome. Using data extracted from the books themselves, we welcome you to visualize a trip one may have taken using these books. Learn More.
